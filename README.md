# Todo App Project

Aplicación de gestión de tareas compuesta por **Backend (API)**, **Frontend Web** y **Aplicación Mobile**.

## 📁 Estructura del proyecto

```
final2
│
├── todo-app-api      # Backend API con Node.js y Express
├── todo-frontend     # Aplicación web (Frontend)
└── todo-mobile       # Aplicación móvil
```

---

# ⚙️ Tecnologías utilizadas

### Backend

* Node.js
* Express
* MySQL
* REST API

### Frontend

* React / Next.js
* JavaScript / TypeScript
* Fetch / API requests

### Mobile

* React Native / Expo

# 🔧 Backend (API)

Entrar a la carpeta del backend:

```bash
cd todo-app-api
```

Instalar dependencias:

```bash
npm install
```

Ejecutar el servidor:

```bash
npm start
```

El servidor correrá en:

```
http://localhost:3000
```

---

# 💻 Frontend

# 📡 Endpoints principales

| Método | Endpoint   | Descripción      |
| ------ | ---------- | ---------------- |
| GET    | /tasks     | Obtener tareas   |
| POST   | /tasks     | Crear tarea      |
| PUT    | /tasks/:id | Actualizar tarea |
| DELETE | /tasks/:id | Eliminar tarea   |
